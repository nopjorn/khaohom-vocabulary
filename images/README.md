# โฟลเดอร์รูปภาพ

ค่าเริ่มต้นเกมนี้ใช้ **emoji** แสดงแทนคำศัพท์ทั้งหมด (ไม่ต้องอัปโหลดรูป)

ถ้าอยากใช้รูปจริงแทน emoji สำหรับคำศัพท์ไหน ทำตามนี้:

1. สร้างโฟลเดอร์ย่อยตามหมวดหมู่ เช่น `images/animals/dog.png` (รองรับ `.svg` ด้วย เช่น `images/animals/dog.svg`)
2. เปิดไฟล์ `../words.js` แล้วหาคำศัพท์ที่ต้องการ เช่น
   ```js
   { emoji: "🐶", th: "หมา", en: "Dog", category: "animals", image: null }
   ```
3. เปลี่ยนค่า `image` เป็น path ของไฟล์รูป:
   ```js
   { emoji: "🐶", th: "หมา", en: "Dog", category: "animals", image: "images/animals/dog.png" }
   ```
4. บันทึกไฟล์ — เกมจะใช้รูปนั้นแทน emoji ให้อัตโนมัติ ไม่ต้องแก้โค้ดส่วนอื่น

แนะนำให้ใช้รูปสี่เหลี่ยมจัตุรัส ขนาดไม่เกิน ~500x500px เพื่อความเร็วในการโหลด

## หลายรูปต่อคำ (สุ่มไม่ให้ซ้ำ)

ถ้าอยากให้คำเดิม (เช่น "หมา") มีได้หลายรูป แล้วเกมสุ่มแสดงคนละรูปในแต่ละครั้งที่เจอคำนั้น ให้ใช้ฟิลด์ `images` (array) แทน `image` (string เดี่ยว):

```js
{ emoji: "🐶", th: "หมา", en: "Dog", category: "animals", image: null, images: [
  "images/animals/dog1.png",
  "images/animals/dog2.png",
  "images/animals/dog3.png",
] }
```

ทุกครั้งที่คำนี้ถูกสุ่มขึ้นมาเป็นคำถาม เกมจะสุ่มเลือกรูปหนึ่งใน `images` มาแสดง ถ้าไม่มี `images` ระบบจะ fallback ไปใช้ `image` เดิม แล้วถ้าไม่มีทั้งคู่ก็ใช้ emoji แทน
